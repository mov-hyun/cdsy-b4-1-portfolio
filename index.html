About

기본기를 설명할 수 있는 개발을 목표로 합니다.

이 포트폴리오는 라이브러리 없이 브라우저가 제공하는 기본 기능만으로 만들었습니다. 시맨틱 HTML로 문서 구조를 세우고, CSS Flexbox와 Grid로 반응형 레이아웃을 구성했습니다.

JavaScript에서는 사용자 입력과 클릭 이벤트를 상태 변경으로 연결하고, 변경된 상태를 기준으로 필요한 DOM만 다시 렌더링합니다.

  • DOM 선택과 이벤트 연결
  • 비동기 API 호출과 상태별 UI
  • 접근성을 고려한 폼과 네비게이션
